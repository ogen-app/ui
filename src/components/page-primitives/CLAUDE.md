# Page Primitives

Reusable page-level building blocks for consistent layouts, loading states, error states, and empty states.

## Files

| File | Description |
|------|-------------|
| `PageContainer.tsx` | Page wrapper with layout variants: `default`, `fullFlex`, `fullscreen` |
| `PageHeader.tsx` | Page header with title, subtitle, and collapsible sidebar toggle |
| `PageLoader.tsx` | Loading state with spinner and message |
| `PageError.tsx` | Error state with customizable header, message, action button, and footer |
| `PageEmpty.tsx` | Empty state with icon, header, message, and optional action buttons |
| `PageContentStub.tsx` | "Not shipped yet" placeholder page with logo quiz game |
| `PageStatusFooter.tsx` | Decorative footer with repeated message text and background texture |
| `PortfolioEmptyState.tsx` | Empty portfolio state with "Add Symbol" button |
| `PortfolioFilterOutState.tsx` | State shown when all holdings are filtered out |
| `PortfolioTBDState.tsx` | "Coming soon" placeholder for unimplemented portfolio features |
